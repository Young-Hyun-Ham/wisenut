# clt-chatbot 기능 정리

## 개요
clt-chatbot은 시나리오 기반 대화와 LLM 기반 자유 대화를 결합한 하이브리드 챗봇 프론트엔드(Next.js)이다.

## 핵심 기능
- 하이브리드 채팅: 시나리오(정형) + LLM(비정형) 응답 결합
- 실시간 대화 및 기록 저장: 대화/시나리오 상태를 실시간으로 저장
- 사용자 인증: dev-login 기반 토큰 인증
- 대화 히스토리/검색: 대화 목록과 전체 메시지 검색
- 시나리오 엔진: 노드/엣지 기반의 대화 흐름 실행
- 다국어 지원: 한국어/영어 (확장 가능)
- 사용자 설정: 테마(라이트/다크), 폰트 크기 설정
- 알림/토스트: 이벤트 알림 및 이력 관리
- 관리자 설정: 즐겨찾기 개수, LLM 공급자(Flowise), Flowise URL 등

## 시나리오 엔진 기능 (노드 타입)
- message: 텍스트 메시지 출력
- slotfilling: 입력을 슬롯에 저장(유효성 검사 포함)
- branch: 조건 분기 또는 버튼 선택 분기
- form: 다중 입력 폼 수집
- api: 외부 API 호출 후 응답 매핑
- llm: LLM 호출 및 결과/슬롯 저장
- toast: 사용자 알림
- iframe: 외부 페이지 임베드
- link: 링크 제공
- setSlot: 슬롯 값 설정/갱신
- delay: 지연 실행

## 관련 화면/영역
- 사용자 채팅 화면
- 대화 히스토리/검색
- 관리자: 시나리오 에디터, 일반 설정
