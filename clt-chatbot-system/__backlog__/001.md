# 제품화 전환 업무 정리

## 배경
- `reference/clt-chatbot`, `reference/react-flow`는 고객사 챗봇 PoC를 위해 사용됨
- PoC는 성공적으로 종료
- 제품화를 위해 소스 개선이 필요

## 대상 구성
- `chatbot-front`: `reference/clt-chatbot` 활용
- `chatbot-backend`: 신규 개발
- `chatbot-admin`: `reference/react-flow` 활용 (추가 개발 필요)
- `chatbot-admin-backend`: 신규 개발

## 핵심 지시사항
- `reference` 하위 두 프로젝트를 분석해 외부 의존성 제거 분석서 작성
- `reference/clt-chat_req_res.md`는 메인 채팅 요청/응답 캡처

## PM 의견/기준
