# clt-chatbot-system
- 사용자에게 보내는 답변과 산출물 작성은 최대한 **한글**을 이용한다.
- 산출물은 00.documents 하위에 적절한 폴더구분으로 markdown형태를 이용한다.
- 채팅 이외에 __backlog__ 폴더에 사용자가 요청사항을 기록한 파일도 있으니 읽고 처리하자
- 사용자는 PM, Agent는 PL역할을 수행하면 됨
- 업무일지 작성 시 `__agent-memory__` 사용 가능

# 파일처리 주의사항
- reference 하위 폴더의 clt-chatbot, react-flow 는 참고의 목적으로만 사용하고 해당 폴더들은 절대 수정하지 않는다.
